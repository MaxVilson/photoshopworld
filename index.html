Описания терминов по Фотошопу

Red Eye Tool — инструмент для удаления эффекта красных глаз человека, который получается в результате фотосъёмки со вспышкой. Принцип работы инструмента заключается в создании выделенной области вокруг красного глаза. Также можно работать одиночными кликами. <..>

Patch Tool — инструмент клонирования при помощи выделенной области. Принцип работы заключается в том, что сначала нужно создать выделенную область и потом переместить её в то место, которое нужно заполнить выделенным участком, либо в то место, которое должно заполнить выделенный участок (в зависимости от параметров инструмента). <..>
